# Gestor-FCE

Gestor-FCE é um software de **gestão de fluxo de caixa empresarial** que utiliza Inteligência Artificial para facilitar o controle financeiro, previsão de receitas e despesas, e otimização da tomada de decisões financeiras.

## Principais Funcionalidades

- **Cadastro e Gerenciamento de Entradas e Saídas:** Controle completo sobre todas as movimentações financeiras.
- **Dashboards e Relatórios:** Visualização intuitiva dos saldos, previsões e tendências do fluxo de caixa.
- **Análise Preditiva com IA:** Geração de previsões automáticas de fluxo de caixa baseada em dados históricos usando modelos de Machine Learning.
- **Alertas Inteligentes:** Sinalização automática de riscos tais como baixas de saldo, contas a pagar/receber e oportunidades de investimento.
- **Multiusuário:** Permite colaboração de equipes com permissões diferenciadas.
- **Exportação e Integração:** Exportação de relatórios e integração com outros sistemas financeiros via API ou planilhas.

## Tecnologias Utilizadas

- **Backend:** Python
- **Frontend:** Angular
- **Containerização:** Docker
- **Inteligência Artificial:** Frameworks Python para Machine Learning.
